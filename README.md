Entorno is a 2D platformer game. It's my project for a game development course at FMI.  

Controls:  
move - WASD  
jump - Space  
pet cat - P  
turn to square - Q  
sleep to next day - N  

Gameplay of v1.0: https://youtu.be/tRwpfBMca2c
